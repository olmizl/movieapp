# WMT ( WHAT IS MOVIE TODAY )

오늘의 인기 영화를 선별하여 보여주는 사이트입니다

## 주요 기능

(1) TMDB의 API를 이용하여 데이터 실시간 연동 <br/>
(2) REACT로 제작하여 새로고침 없이 페이지 렌더링 <br/>
(3) 검색기능을 이용하여 선별된 영화 외에도 영화 검색 가능
